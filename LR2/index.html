История HTML, CSS и JavaScript

История HTML

  • Изобретен Тимом Бернерсом-Ли в 1991 году в CERN.
  • Первая версия в 1993 году.
  • Множество версий, включая HTML5 в 2014 году.

История CSS

  1. Предложен Хоконом Виумом Ли в 1994 году.
  2. Первый стандарт CSS1 в 1996 году.
  3. Разработан для разделения стиля от содержания.

История JavaScript

1995 год
Изобретен Бренданом Эйхом для Netscape.
1997 год
Стал стандартом ECMA-262.
Современность
